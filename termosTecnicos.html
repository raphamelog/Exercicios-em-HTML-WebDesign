Termos Técnicos

HTML
Linguagem de marcação usada para estruturar páginas web.
CSS
Linguagem de estilo usada para definir a aparência das páginas web.
JS
Linguagem de programação usada para adicionar interatividade às páginas web.

HTML

HTML (HyperText Markup Language) é a linguagem padrão para criar páginas web. Ela permite estruturar o conteúdo usando elementos como títulos, parágrafos, listas, links e muito mais.

CSS

CSS (Cascading Style Sheets) é uma linguagem de estilo utilizada para definir cores, fontes, espaçamentos e o layout das páginas web, tornando-as mais atraentes visualmente.

JS

JS (JavaScript) é uma linguagem de programação que permite criar páginas web dinâmicas e interativas, manipulando elementos da página e respondendo a eventos do usuário.
